Rubric Editor
▶ How to use
Building the rubric
  1. 1.Fill in the rubric name and any criteria. Click a criterion or achievement level name to edit it in place.
  2. 2.Type descriptors in the cells. Press Enter to add a new bullet/element, or use the + bullet button.
  3. 3.Press Tab to move to the next achievement level. Shift+Tab moves back.
  4. 4.To copy a cell across levels, hover the cell and click Copy, then click Paste on the target — or use ⌘⇧C / ⌘⇧V (Ctrl on Windows) with a cell focused.
Importing into Compass
  1. 1.Click ↓ Export JSON and save the downloaded file.
  2. 2.In Compass: School Resources → Subject folder → Add Content → Import Rubric.
  3. 3.Select the downloaded file and test the rubric before use.
Saving your work
  1. 1.This editor does not save automatically. To preserve a draft, click ↓ Export JSON and keep the file.
  2. 2.To resume later, click ↑ Load JSON and paste or select the saved file.
Note: progress will be lost if you close or refresh the page without exporting first.
